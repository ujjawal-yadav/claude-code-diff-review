# Code of Conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). The short version:

- Be respectful. Assume good faith. Disagree about ideas, not people.
- Welcoming and inclusive language. No harassment, no discriminatory jokes, no personal attacks.
- Constructive feedback only. "This code is bad" is unhelpful; "This pattern has issue X because Y" is.
- Privacy: never publish someone else's private information without consent.

## Reporting

Project maintainer contact: file a [security advisory](https://github.com/ujjawal-yadav/claude-code-diff-review/security/advisories/new) (yes, even for CoC matters — that's the most private channel).

All reports are confidential. Maintainers will review and respond within 7 days.

## Enforcement

Maintainers reserve the right to remove comments, close issues, and ban contributors who violate this Code of Conduct. The same standard applies to everyone, including maintainers.
